X~N(μ=1.2, σ=0.15)

$Z = \frac{x - \mu}{\sigma}$

當 $x=1$時， $z_1 = \frac{1 - 1.2}{0.15}$ = -1.3333

當 $x=1.10$時， $z_2 = \frac{1.10 - 1.2}{0.15}$ = -0.6666

### 查表得

$P(z<-1.33)$ : 0.0918

$P(z<-0.67)$ : 0.2514

### 計算

$P(1.00 < x < 1.10) = P( z <-0.67) - P(z < -1.33) = 0.2514 - 0.0918 = 0.1596$
