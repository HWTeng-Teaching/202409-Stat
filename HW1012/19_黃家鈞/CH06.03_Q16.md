$Binominal(n,p) \approx N(\mu = np, \sigma^2 = npq)$

Normal Approximation:
1. n =50, p = .47
2. np = 23.5 and nq = n(1-p) = 26.5, np >5 and nq >5
3. $\mu = np = 23.5$ and $\sigma^2 = npq = 12.455$ and$\sqrt{npq} = 3.53$
4. 5. $z = \frac{x \pm 0.5 - np}{\sqrt{npq}}$

a.

透過連續校正 $P (X < 30)  \approx P(  z < \frac{29.5-23.5}{3.53} ) = P(  z \leq 1.70 ) = 0.9554$

c.
