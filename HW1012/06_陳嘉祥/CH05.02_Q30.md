### a. $P(X < 12) \text{ for } n = 20, p = 0.5$ 查表得到 :  

$\{
P(X < 12)=
P(X \le 11)
\approx 0.748
\}$

### b. $P(X \le 6) \text{ for } n = 15, p = 0.4$ 查表得到 :   

$\{
P(X \le 6)
\approx 0.610
\}$

### c. $P(X > 4) \text{ for } n = 10, p = 0.4$ 查表得到 : 

$\{
P(X > 4)=
1-P(X \le 4)=
1-0.633
\approx 0.367
\}$

### d. $P(X \lg 6) \text{ for } n = 15, p = 0.6$ 查表得到 :  

$\{
P(X \ge 6)=
1-P(X \le 5)=
1-0.034
\approx 0.966
\}$


### e. $P(3 < x < 7) \text{ for } n = 10, p = 0.5$ 查表得到 :  

$\{
P(3<X<7)=
P(X \le 6)-P(X \le 3)=
0.828-0.172
\approx 0.656
\}$
