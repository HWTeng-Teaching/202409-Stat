without replacement

| Possible samples  |   x̄ |
|----------------   |---- | 
| (10, 15)          |12.5 |
| (10, 21)          |15.5 |
| (10, 22)          |16.0 |
| (15, 21)          |18.0 |
| (15, 22)          |18.5 |
| (21, 22)          |21.5 |
