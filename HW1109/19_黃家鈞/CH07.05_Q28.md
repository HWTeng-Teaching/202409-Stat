p = 0.7 ， n = 50

根據中央極限定理(CLT)

$\mu_{\hat{p}} = p = 0.7$

$\sigma_{\hat{p}} = \sqrt{\frac{p(1-p)}{n}} = 0.0648$

$P(\hat{p} > 0.8) = P(Z > \frac{0.8 - 0.7}{0.0648}) = P(Z > 1.54) = 1 - P(Z ≤ 1.54) = 1- 0.9382 = 0.0618 $(查表得)
