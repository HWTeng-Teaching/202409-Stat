抽後不放回的情況

|  $$\bar{x}$$ |P( $$\bar{x}$$ ) |
|-------------:|:---------------:|
| 12.5         | $$\frac{1}{6}$$ |
| 15.5         | $$\frac{1}{6}$$ |
| 16.0         | $$\frac{1}{6}$$ |
| 18.0         | $$\frac{1}{6}$$ |
| 18.5         | $$\frac{1}{6}$$ |
| 21.5         | $$\frac{1}{6}$$ |

E( $$\bar{x}$$ ) = 12.5 * 1/6 + 15.5 * 1/6 +16 * 1/6 + 18 * 1/6 + 18.5 * 1 /6 + 21.5 * 1/6 = 17


抽後放回的情況

|  $$\bar{x}$$ |P( $$\bar{x}$$ ) |
|-------------:|:---------------:|
| 10.0         | $$\frac{1}{16}$$ |
| 12.5         | $$\frac{2}{16}$$ |
| 15.0         | $$\frac{1}{16}$$ |
| 15.5         | $$\frac{2}{16}$$ |
| 16.0         | $$\frac{2}{16}$$ |
| 18.0         | $$\frac{2}{16}$$ |
| 18.5         | $$\frac{2}{16}$$ |
| 21.0         | $$\frac{1}{16}$$ |
| 21.5         | $$\frac{2}{16}$$ |
| 22.0         | $$\frac{1}{16}$$ |

E( $$\bar{x}$$ ) = 10 * 1/16 + 12.5 * 2/16 + 15 * 1/16 + 15.5 * 2/16 ..... 22 * 1/16  = 17

無論是抽後放回或不放回，mean of $$\bar{x}$$ 都是17 ( E( $$\bar{x}$$ )=17 )
